O Que é HTML?

O Que é HTML? HTML ( abreviação de Hypertext Markup Language) é uma linguagem de marcação utilizada na estruturação de páginas web. Sua sintaxe é bastante simples e, assim como a XML é baseada em tags, que representam os diversos elementos de uma página, como imagens e links. Selecionaie algumas TAGS para tentar entender melhor. Tag Html

↖-Voltar-

O marcador < HTML > representa a raiz de um HTML, ou seja, todos os outros elementos devem ser escritos após esse elemento.

Tag Head

↖-Voltar-

O marcador < Head > providencia informações gerais sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

Tag Link

↖-Voltar-

O marcador < Link > especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.

Tag Meta

↖-Voltar-

O marcador < Meta > define qualquer informação de metadados que não podem ser definidos por outros elementos.

Tag Style

↖-Voltar-

O marcador < Style > contém informações de estilo para um documento ou uma parte do documento. As informações de estilo geralmente estão no arquivo CSS do site.

Tag Title

↖-Voltar-

O marcador < Title > define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

Tag Sub / Sup

↖-Voltar-

O marcador < Sub / Sup > "Superscript Element" ou sobrescrito é utilizado escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo.
